# IT Support & Help Desk Labs

This repository contains hands-on labs I created to practice real-world IT support and help desk skills:
troubleshooting, documenting issues, and thinking like frontline support.

## Labs

- [Lab 01 – Email Login Troubleshooting](./Email/email-login-troubleshooting.md)  
  Simulated a user who couldn’t sign into their email on a MacBook; documented questions, diagnostic steps, root cause, and resolution.

- [Lab 02 – macOS New User Setup & Hardening](./MacOS/new-user-setup.md)  
  Simulated setting up a new employee’s Mac with security and usability best practices.
